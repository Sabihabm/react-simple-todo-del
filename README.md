# 📝 Simple Todo App

A basic and responsive Todo application built using React.
This delete the compeleted task.

---

## 🚀 Features

* 📋 Display list of todos
* ❌ Delete tasks
* ⚡ Fast and simple UI

---

## 🛠️ Tech Stack

* React
* JavaScript
* HTML
* CSS

---

## 📂 Project Setup

1. Clone the repository

```
git clone https://github.com/your-username/react-simple-todo-del.git
```

2. Navigate to project folder

```
cd react-simple-todo-del
```

3. Install dependencies

```
npm install
```

4. Run the project

```
npm run dev
```
